
# TP - Gestión Colaborativa, Control de Versiones y Organización Empresarial

## Descripción

Este proyecto fue realizado para la materia Organización Empresarial de la UTN.

El objetivo del trabajo fue aplicar herramientas de control de versiones y gestión colaborativa utilizando Git, GitHub y Jira.

## Escenario elegido

Análisis de ventas de una pequeña empresa.

## Tecnologías utilizadas

- Python
- Pandas
- Matplotlib
- Git
- GitHub
- Jira
- Google Colab

## Estructura del proyecto

```text
datos/
scripts/
resultados/
```

## Funcionalidades

- Lectura de dataset CSV.
- Cálculo de ventas totales.
- Análisis por producto.
- Generación de gráfico de ventas.
